## üé® Image Guidelines

### Recommended Formats
- **Screenshots**: PNG (high quality, transparent backgrounds when needed)
- **Diagrams**: JPEG/PNG (clear, high resolution)
- **Charts**: PNG (crisp lines and text)

### Image Specifications
- **Minimum Resolution**: 1920x1080 for hero images
- **Chart Screenshots**: 1200x800 minimum
- **Diagrams**: Vector-based when possible (exported as high-res PNG)

## üì∏ Creating Portfolio Images

### Dashboard Screenshots
1. Open the live demo at https://awap-next-app.onrender.com
2. Navigate to the relevant dashboard view
3. Use browser developer tools to set viewport to 1920x1080
4. Capture screenshot (F12 > Device toolbar > Screenshot)
5. Crop and optimize in image editor

### Architecture Diagrams
- Use tools like: Draw.io, Lucidchart, Figma, or Excalidraw
- Export as high-resolution PNG or SVG
- Include clear labels and color-coded components

### Data Visualizations
- Capture directly from the application
- Ensure data is clearly visible
- Include axis labels and legends
- Use consistent color schemes

1. **Consistency**: Use similar color schemes across all images
2. **Clarity**: Ensure text is readable at various sizes

4. **Quality**: Never use blurry or pixelated images
5. **Branding**: Maintain consistent styling that reflects the SheDev brand
